## Table of contents

## Installation

```
npm install --save next-gtm

or

yarn add next-gtm

```

## Usage

Your Layout Component:

```javascript
import Gtm from 'next-gtm'

<Head>
// layout code
</Head>

<Gtm id='GTM-XXXXXXXX'/>
// ...

```
